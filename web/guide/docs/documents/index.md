# 『叶子书摘』使用指南

## **一、简介**

### **叶子书摘是什么？**

叶子书摘的理念是“重温比记录更有意义”，专注于书摘回顾和记录。

### **特色**

1. 弱化书籍概念，书籍标签化，方便筛选，快速回顾。
2. 漫步功能，随机回顾之前的书摘，灵感再现。
3. 简化记录操作，文字、图片、OCR 多种记录方式，不打断阅读心流。

## **二、功能介绍**

### **书摘浏览**

按照时间流进行书摘浏览；通过筛选可以筛选对应的书籍。
